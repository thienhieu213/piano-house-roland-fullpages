RP701 / F701 / RP107 / F107 System Program (Ver.1.07)

HOW TO KNOW THE VERSION
Before you start, check the system program version of your RP701 / F701 / RP107 / F107.
* RP701 / F701 and RP107 / F107 have different procedures of checking versions.
 
RP701 / F701
  1. Press the [] (settings) button.
  2. Turn the [] ( select/confirm ) knob to select "Version."
  3. Press the [] ( select/confirm ) knob.
    The version information is displayed.
    If this shows the latest version (version 1.06 or later), there's no need to update.
  4. Press the [] (settings) button.
 
RP107 / F107
  1. Hold down [] (Volume + ) and switch on the power to the unit.
    * Continue to hold down the power button until the topmost indicator to the right of the volume buttons flashes.
  2. Examine the pattern of lighted and dark indicators to determine the current version.
    If the indicators are not lighted and dark as described below, perform an update.
    * If the pattern of lighted and dark is as follows, the system is at the latest version. No update operation is necessary.
    • 1st from top (topmost) : Flashing
    • 2nd from top : Lighted
    • 3rd from top : Dark
    • 4th from top : Dark
    • 5th from top : Dark
  3. After verifying the displayed version, switch off the power.

WHAT YOU WILL NEED TO UPDATE YOUR RP701 / F701 / RP107 / F107
  • RP701 / F701 / RP107 / F107
  • USB Memory
    * The USB Memory need to be formatted on a RP701 / F701 or PC.
  • A computer with a USB connector ( Windows, Mac )
  • Update file downloaded from this page
CAUTIONS WITH THE UPDATE
  • Never power-off your while the update is in progress!
    If a power failure or similar accident occurs during the update process, the won't be able to start up in normal operating mode. This will require servicing, so you will need to contact Roland service center near you if this occurs.
  • We regret that we are unable to answer questions regarding the update procedure using this system program. Please perform the update responsibly, following the directions given in this document.
  • RP701 / F701 has the ability to save your user data to a USB memory. If a problem occurs and your user data is lost, you will be able to restore the original state by reloading the data you saved. For details on the procedure, refer to the following sections of the Owner's Manual.
    • Initializing the Memory (Format Media)
      * This formats the USB memory device.
    • Saving Piano Settings (Piano Setup Export)
    • Loading Piano Settings (Piano Setup Import)
    • Recording Your Keyboard Performance - Copying a recorded song
    * Please be aware that "Ambience Type" is not saved to the USB memory device.
GETTING THE UPDATE FILE
Download the file named "rp701_f701_sys_v107.zip".
The downloaded file is an archive file in zip format. Right-click to select Extract All, and then follow the instructions. After expanding the file, the folder "rp701_f701_sys_v107" holding the file "R079_UPA.BIN" will be appeared.
PROCEDURE FOR THE UPDATE
 
Creating the Update USB Memory
  1. Copy update files R079_UPA.BIN to the root directory of a USB Memory.

Update Procedure
* The RP701 / F701 and RP107 / F107 have different update procedures.
 
RP701 / F701
  1. Make sure the unit's power is switched off, then connect the USB memory into the USB MEMORY port.
  2. Hold down [] (settings) button and switch on the power to the unit.
    Continue to hold down the [] (settings) button until " ===update=== " appears on the display.
  3. When the message " = update finished. = " appears, the update operation has finished.
    * The update operation takes about 2 minutes.
  4. Press and hold the power button to turn off the power.
    Detach the USB Memory from the digital piano.

    This completes the update operation. The new system program runs at the next startup.
 
RP107 / F107
  1. Turn off the power to the RP107 / F107 and connect the USB memory device to the unit's USB Memory port.
  2. Hold down [] (settings) button and turn on the power to the digital piano.
    The update starts. While the update is in progress, the indicators to the right of the volume buttons repeatedly flash and go dark in order from the bottom.
    * The update operation takes about 2 minutes.
    * Never switch off the power or detach the USB memory device while the update is in progress.
  3. When the indicators to the right of the volume buttons all flash simultaneously, the update is complete.
    Turn off the power to the RP107 /F107.
  4. Detach the USB memory from the RP107 /F107.
