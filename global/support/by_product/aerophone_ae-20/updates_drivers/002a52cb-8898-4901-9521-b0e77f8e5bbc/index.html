Aerophone AE-20 System Program ( Ver.1.12 )

HOW TO FIND THE VERSION
Please check the current version of the system working on your AE-20 before downloading the file.
 
  1. Turn the AE-20's switch "ON".
  2. On the scene screen, press the [MENU] button.
    The Menu screen appears.
  3. Use the [-] [+] (< / > ) buttons to select "Version".
    The current program version is shown.
    • Ver.1.10 or later: Applying this update is not necessary.
    • Ver.1.02 or earlier: This is an old version. Please update to the latest version.
  4. Press the [SCENE] button or turn the [SCENE CATEGORY] knob to return to the scene screen.
UPDATE HISTORY
[Ver.1.12] JAN 2023
We updated only the version number for administrative purposes.
No changes affect product specifications.
[Ver.1.11] OCT 2022
The factory-default test program was updated.
Actual product operation is unaffected.
[Ver.1.10] SEP 2022
Functionality Improvements
  • Improvements to the following functions in the "Sax" and "Clarinet" fingering modes
    • Unintended notes are now more difficult to produce even when the incorrect key is fingered.
    • The sound-production behavior of keys played in tandem on acoustic instruments has been partially re-created.
  • It is now possible to access functions that use a USB flash drive even while a performance key is depressed.
  • When the Harmony function is on, the sequence in which the Note On messages for harmony notes is sent has been improved.
[Ver.1.02] JUN 2022
The factory-default test program was updated.
Actual product operation is unaffected.
WHAT YOU NEED FOR THE UPDATE
Updating the AE-20 system program requires the following items.
 
  • Aerophone AE-20
  • Included AC adaptor
  • USB flash drive (Type-C, FAT32 file system)
    *If the AE-20 fails to recognize a USB flash drive, then on a computer, format the USB flash drive to use the FAT32 file system (on a Mac: "MS-DOS (FAT)").
    *Using a USB Type-C adapter lets you connect a USB flash drive that has a USB A connector. Note, however, that making a good connection when using a multifunction-type adapter is sometimes not possible.
  • PC (Windows/Mac) to write the updater to the USB Flash Memory
CAUTION WHEN PERFORMING THE UPDATE
 
  • Never apply this update data to any product other than the AE-20. This update data may destroy the data or system program of other products.
  • Never power-off your AE-20 while the update is in progress!
    If a power failure or similar accident occurs during the update process, the AE-20 won't be able to start up in normal operating mode. This will require servicing, so you will need to contact Roland service center near you if this occurs.
  • We regret that we are unable to answer questions regarding the update procedure using this system program.
DOWNLOADING AND UNARCHIVING THE COMPRESSED FILE
 
  1. Read the software license agreement at the bottom of the screen and select the "I agree" check box.
  2. Click the "Download" button to download the system program (ae20_sys_v112.zip).
  3. Unarchive the downloaded file.
    • Windows: Right-click the file and select "Extract All," then click "Extract".
    • macOS: Double-click the file to unzip it.
      *Never double-click "AE-20_UPA_up.bin" displayed inside the folder after unarchiving.
  4. Copy "AE-20_UPA_up.bin" to the level displayed immediately on accessing the USB flash drive (the USB flash drive's root directory).
    • Windows
    • macOS
PROCEDURE FOR THE UPDATE
Caution
 
  • The final step of the update procedure is an operation to return the system settings to their factory-default state (a factory reset). If it will be necessary to restore system settings after updating, make a written note of the required parameter values.
    *Parameter types and setting-value levels might be changed by this update.
    *Performing this operation does not erase the scenes that are stored in the user scenes.
  • You must not rename the extracted file.
  • Never turn off the AE-20's power while the update is in progress. Otherwise the system program or the AE-20 itself may be destroyed.

Follow the steps below to install the extracted system program on your AE-20.
 
  1. Turn the AE-20's switch "OFF".
  2. Connect the USB flash drive containing the "AE-20_UPA_up.bin" file to the AE-20.
  3. Connect the included AC adapter to the AE-20.
  4. Hold down the +2 octave and -2 octave keys simultaneously and switch on the power to the unit.

    The display shows "AE-20 Updater", and the update starts.
    *It takes about 3 minutes to complete the update.
  5. Once the message "Please Power Off" appears on the display, turn the AE-20 off.
  6. Remove the USB flash drive from the AE-20.
  7. Turn the AE-20's switch "ON".
    NOTE: The message "Upd Sub Cpu Main" might be displayed, and another update starts. When "Turn off power" is displayed, turn the power back on and proceed to the next operation.
  8. Press the [MENU] button.
    The Menu screen appears.
  9. Use the [-] [+] (< / >) buttons to select "Factory Reset," and press the [MENU] button.
    A confirmation message appears.
  10. Press the [+] (>) button.
    A confirmation message appears again.
  11. Press the [MENU] button.
    The display indicates "Executing...," and the factory reset operation is executed.
  12. Once the message "Turn off the power" appears on the display, turn the AE-20 off and then on again.
    *If necessary, change the system-setting parameters to the values previously written down.
