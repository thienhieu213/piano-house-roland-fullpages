JUNO-X System Program (Ver.1.11)

HOW TO TELL THE VERSION
Please check the version of the system working on your JUNO-X before downloading the archive file.
  1. Turn on the power to the unit.
  2. Press the [MENU] (WRITE) button.
    The MENU screen apears.
  3. Use the [SELECT] knob to select “INFORMATION,” and then press the [ENTER] (INIT) button.
    The version information appears.

WHAT YOU WILL NEED TO UPDATE YOUR JUNO-X
  • JUNO-X
  • USB flash drive (The USB Memory need to be formatted on a JUNO-X.)
CAUTION WHEN PERFORMING THE UPDATE
  • Never apply this update data to any product other than the JUNO-X. This update data may destroy the data or system program of other products.
  • Never power-off your JUNO-X while the update is in progress!
    If a power failure or similar accident occurs during the update process, the JUNO-X won't be able to start up in normal operating mode. This will require servicing, so you will need to contact Roland service center near you if this occurs.
  • We regret that we are unable to answer questions regarding the update procedure using this system program.
DOWNLOADING AND UNARCHIVING THE COMPRESSED FILE
  1. Read the software license agreement at the bottom of the screen and select the "I agree" check box.
  2. Click the "Download" button to download the system program (junox_sys_v111.zip).
  3. Unarchive the downloaded file.
    • Windows: Right-click the file and select "Extract All," then click "Extract".
    • macOS: Double-click the file to unzip it.
      - Never double-click "AE-20_UPA_up.bin" displayed inside the folder after unarchiving.
  4. Copy "JUNOX_UPA_up.bin" to the level displayed immediately on accessing the USB flash drive (the USB flash drive's root directory).
PROCEDURE FOR THE UPDATE
  1. Turn the power of the JUNO-X off.
  2. Connect the USB flash drive in which you saved the "JUNOX_UPA_up.bin" file to the JUNO-X.
  3. While pressing the [MENU] (WRITE) button, turn on the power.
    "JUNO-X Updater..." appears on the screen and the update starts automatically.
    - Never turn off the power or disconnect the USB flash drive while the update is in progress. Doing so might make the unit impossible to start.
  4. When the display indicates "Please power off," turn the JUNO-X's power off.
  5. Disconnect the USB flash drive from the JUNO-X.
    The update operation is finished. The new system program will be run at the next startup. Check to make sure the version has been updated.
