VERSELAB MV-1 System Program ( Ver.1.81 )

HOW TO TELL THE VERSION
Before you start, check the system program version of your VERSELAB MV-1.
When the power to the VERSELAB MV-1 is turned on, the title screen appears on the display.
The current version is displayed at the bottom right of the title screen.

* Don't perform the update if the product is already up-to-date.
UPDATE HISTORY
[ Ver.1.81 ] NOV 2022
Bug Fixes
The following bugs were fixed.
  • SDZ files cannot be loaded when there are more than 33 SDZ/SVZ files.
  • SDZ files cannot be loaded when more than 33 SDZ/SVZ files are loaded in a row.
  • Drum Kit of SDZ files cannot be loaded.
  • Arpeggio may not work correctly when MIDI Channel is changed.
  • Wave file loading may not work correctly on Vocal Track.

WHAT YOU WILL NEED TO UPDATE YOUR VERSELAB MV-1
  • VERSELAB MV-1
  • A computer with a SD Card Slot ( Windows, Mac )
  • SD Card formatted on the VERSELAB MV-1
    For information on how to format the SD card, go to [ Owner's Manuals ] and refer to "VERSELAB MV-1 Owner's Manual -- Formatting an SD Card."
CAUTION WHEN PERFORMING THE UPDATE
  • Never apply this update data to any product other than the VERSELAB MV-1. This update data may destroy the data or system program of other products.
  • Never power-off your VERSELAB MV-1 while the update is in progress!
    If a power failure or similar accident occurs during the update process, the VERSELAB MV-1 won’t be able to start up in normal operating mode. This will require servicing, so you will need to contact Roland service center near you if this occurs.
  • We regret that we are unable to answer questions regarding the update procedure using this system program. Please perform the update responsibly, following the directions given in this document.
GETTING THE UPDATE FILE
Please open/extract the downloaded "mv1_sys_v181.zip" file on your computer to get the update file in the "mv1_sys_v181" folder.
PROCEDURE FOR THE UPDATE
Caution
  • You must not rename the extracted file.
  • Never turn off the VERSELAB MV-1's power while the update is in progress. Otherwise the system program or the VERSELAB MV-1 itself may be destroyed.

Please install the extracted system program file into your VERSELAB MV-1 according to the following procedure.
* For information on "Returning to the Factory Settings" procedures, refer to [ Owner's Manual ].

STEP 1: Updating the VERSELAB MV-1 System Program
Use the following procedure to update the VERSELAB MV-1 using and SD card.
  1. Download the "MV1_UPA_up.bin" to you computer.
  2. Copy the "MV1_UPA_up.bin" file to the root folder of the SD card.
    * If you're using a new SD card, format it on the VERSELAB MV-1 first.
  3. Safely remove the SD card from your computer and carefully insert it into the VERSELAB MV-1.
  4. With the unit switched off, hold down the [SHIFT] button and turn the VERSELAB MV-1 on. The update will then start.
  5. The message "COMPLETE" will appear on the display once the update process is complete.
  6. Restart the unit.

STEP 2: Returning to the Factory Settings
* Performing the procedure in "Returning to the Factory Settings (Factory Reset)" initializes the "System Settings" items. We apologize for the inconvenience, but please configure these again.
  1. Hold down the [ SHIFT ] button and press the [ SYSTEM ] button to display the UTILITY screen.
  2. Use the [ VALUE ] dial to select "FACTORY RESET," then press the [ VALUE ] dial.
    A confirmation message appears.
  3. To execute, use the [ VALUE ] dial to select "OK," and press the [ VALUE ] dial.
  4. When the display indicates "Turn off power" turn the VERSELAB MV-1's power off and on again.
