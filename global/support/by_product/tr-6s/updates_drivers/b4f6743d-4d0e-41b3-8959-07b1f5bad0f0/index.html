TR-6S System Program ( Ver.1.51 )

HOW TO TELL THE VERSION
Before you start, check the system program version of your TR-6S.
  1. Press [ UTILITY ] to invoke the menu screen.
  2. Use the [ VALUE ] knob to choose INFORMATION : Version, and then press the [ VALUE ] knob.
    The version information screen appears.

* Don't perform the update if the product is already up-to-date.
UPDATE HISTORY

WHAT YOU WILL NEED TO UPDATE YOUR TR-6S
  • TR-6S
  • USB cable
  • A computer with a USB connector ( Windows, Mac )
  • SD Card formatted on the TR-6S
CAUTION WHEN PERFORMING THE UPDATE
  • Never apply this update data to any product other than the TR-6S. This update data may destroy the data or system program of other products.
  • Never power-off your TR-6S while the update is in progress!
    If a power failure or similar accident occurs during the update process, the TR-6S won't be able to start up in normal operating mode. This will require servicing, so you will need to contact Roland service center near you if this occurs.
  • We regret that we are unable to answer questions regarding the update procedure using this system program. Please perform the update responsibly, following the directions given in this document.
GETTING THE UPDATE FILE
Please open/extract the downloaded "tr6s_sys_v151.zip" file on your computer to get the update file in the "tr6s_sys_v151" folder.
PROCEDURE FOR THE UPDATE
Caution
  • You must not rename the extracted file.
  • Never turn off the TR-6S's power while the update is in progress. Otherwise the system program or the TR-6S itself may be destroyed.

Please install the extracted system program file into your TR-6S according to the following procedure.
* For information on "BACKUP," "FACTORY RESET," and "RESTORE" procedures, refer to [ Owner's Manual ].

STEP 1: Backing Up to SD Card
  1. Press the [ UTILITY ] button.
  2. Use the [ VALUE ] knob to select "UTILITY: Backup."
  3. Press the [ VALUE ] knob.
  4. Edit the name if necessary.
  5. Press the [ VALUE ] knob.
    A confirmation message appears.
  6. To execute, use the [ VALUE ] knob to select "OK," and press the [ VALUE ] knob.

STEP 2: Updating to TR-6S
  1. Make sure no SD card is inserted in the TR-6S's SD card slot.
    * If an SD card is already inserted, then take out the card.
  2. Hold down [ SHIFT ] and switch on the power.
    "==== update ====" appears on the display.
  3. Connect the computer to the TR-6S's USB port by a USB cable.
  4. Open the "TR-6S" drive folder in your personal computer.
  5. Copy TR6S_UP.bin in the "tr6s_sys_v151" folder into the "TR-6S" drive by dragging and dropping it.
    * Do not copy the "tr6s_sys_v151" folder.
  6. Press [ TEMPO ] button.
    The message "Updating..." appears on the display and the update starts.
  7. The message "Update Finished." will appear on the display when the update process completes.
  8. Turn the TR-6S power off.
    Please verify the version number "1.51" with the procedure described in "How to tell the version" part of this document.

    At the initial powerup following an update, the message "Updating Panel" sometimes appears on the display.
    If "Updating Panel" appears, wait until the message "Completed" appears on the display, then reset the power again.
 
If Updating Using an SD Card
You can also update the TR-6S by using an SD card.
* If you are using a SD card to perform the update, carry out the procedure while no USB cable is connected to the TR-6S.

Follow the procedure below to carry out the update.
  1. Format an SD card on the TR-6S.
  2. Put the "TR6S_UP.bin" at the root folder of the SD Card.
  3. Leaving the power to the TR-6S switched off; connect the SD Card to the SD Card port.
  4. Hold down [SHIFT] button and turn on the power.
    The message "Updating..." appears on the display and the update starts.
  5. The message "Update Finished." will appear on the display when the update process completes.
  6. Turn the TR-6S power off.

    At the initial powerup following an update, the message "Updating Panel" sometimes appears on the display.
    If "Updating Panel" appears, wait until the message "Completed" appears on the display, then reset the power again.

STEP 3: Returning to the Factory Settings
  1. Press the [ UTILITY ] button.
  2. Use the [ VALUE ] knob to select "UTILITY: Factory Reset."
  3. Press the [ VALUE ] knob.
  4. Use the [ VALUE ] knob to select "ALL," then press the [ VALUE ] knob.
    A confirmation message appears.
  5. To execute, use the [ VALUE ] knob to select "OK," and press the [ VALUE ] knob.
  6. When the display indicates "Completed. Turn off power." turn the TR-6S's power off and on again.

STEP 4: Restoring Data That Was Backed Up to SD Card
  1. Press the [ UTILITY ] button.
  2. Use the [ VALUE ] knob to select "UTILITY: Restore."
  3. Press the [ VALUE ] knob.
    The RESTORE screen appears.
  4. Use the [ VALUE ] knob to select the file that you want to restore.
  5. Press the [ VALUE ] knob.
    A confirmation message appears.
  6. To execute, use the [ VALUE ] knob to select "OK," and press the [ VALUE ] knob.

STEP 5: Updating the Tone Data
If an update to the TR-6S results in the preloaded Patch/Performance tone data being updated, you can load the latest tone data by executing the procedure in "Returning to the Factory Settings (FACTORY RESET)." In "Returning to the Factory Settings (FACTORY RESET)," you can select the items to be affected by the factory reset. For details, go to [ Owner's Manual ] and refer to "Returning to the Factory Settings (FACTORY RESET)."
* Executing the procedure in "Returning to the Factory Settings (FACTORY RESET)" causes the tone data/settings in the TR-6S to be lost.
Before executing the procedure in "Returning to the Factory Settings (FACTORY RESET)," use the procedure in "Backing Up Data to SD Card (BACKUP)" to save any necessary data onto an SD card.
  1. Press the [ UTILITY ] button and use the cursor buttons to select "Factory Reset," then press [ VALUE ] knob.
  2. Use the [ VALUE ] knob to choose a factory-reset parameter.
    * Please note that when you execute "ALL" or execute "KIT" or "PTN," then depending on the selected parameter, created presets and the settings on the TR-6S are returned to their factory-default state.
    • To return all TR-6S Settings, Kits, and Patterns to their factory defaults, select "ALL."
    • When you want to return Kits to their factory defaults, select "ALL" or "KIT."
    • When you want to return Patterns to their factory defaults, select "ALL" or "PTN."
    • To keep the TR-6S's settings unchanged and return only Kits or Patterns to their factory defaults, select "KIT" or "PTN."
  3. Press the [ VALUE ] knob.
  4. The message "Factory Reset?" is displayed. Press the [ VALUE ] knob button.
    When the factory reset has finished, the message "Completed. Turn off power." is displayed.
  5. Reset the power to the TR-6S.
