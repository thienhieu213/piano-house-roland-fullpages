TR-8 System Program (Ver.1.61)

How to tell the version
Before you start, check the system program version of your TR-8.
  1. Hold down Pad [ 14 ] and [ 16 ], and switch on the power.
    The [ START/STOP ] button blink.
  2. Press the [ START/STOP ] button.
    The current version is displayed with three digits on the display.
    ( When Version 1.61 is used: 161 )
  3. After checking version of the product, power off the TR-8 without performing any button operation.
* Don't perform the update if the product is already up-to-date.
 
Update history
[ Ver.1.61 ] JAN 2019
Additional Function
  • Serato DJ Pro Ver.2.1.1 or later is now supported. Synchronized playback with compositions on Serato DJ Pro is possible. ( FEB 2019 )
    Please refer to the "Roland TR / Serato DJ Pro Connection Guide" for detailed information in the [ Owner's Manuals ].

    * Using this functionality requires a DJ controller compatible with Serato DJ Pro.
    * The Roland DJ-202, DJ-505, and DJ-808 cannot be used as the DJ controller.

[ Ver.1.60 ] AUG 2018
Additional Function
Please refer to the "TR-8 Update" for detailed information in the [ Owner's Manuals ].
  • A STEP LOOP function was added.
Bug Fixes
The following bugs were fixed.
  • Minor bugs have been fixed.

[ Ver.1.50 ] AUG 2017
Additional Functions
  • Supported frequencies during a USB connection were expanded to 44.1, 48, and 96 kHz.
    * To make use of 44.1 and 48 kHz, use the latest version of the driver (version 1.5).
  • Functionality for triggering the SP-404SX or SP-404A was added.
Please refer to the "TR-8 Update" for detailed information in the [ Owner's Manuals ].

[ Ver.1.20 ]
Please refer to the "TR-8 Updates" for detailed information in the [ Owner's Manuals ].
 
What you will need to update your TR-8
  • TR-8
  • AC adaptor
  • USB cable
  • A computer with a USB connector ( Windows, Mac )
 
Caution when performing the update
  • Never apply this update data to any product other than the TR-8. This update data may destroy the data or system program of other products.
  • Never power-off your TR-8 while the update is in progress!
    If a power failure or similar accident occurs during the update process, the TR-8 won’t be able to start up in normal operating mode. This will require servicing, so you will need to contact Roland service center near you if this occurs.
  • We regret that we are unable to answer questions regarding the update procedure using this system program. Please perform the update responsibly, following the directions given in this document.
Getting the update file
Please open/extract the downloaded "tr8_sys_v161.zip" file on your computer to get the update file in the "tr8_sys_v161" folder.
Procedure for the update
Caution
  • You must not rename the extracted file.
  • Never turn off the TR-8's power while the update is in progress. Otherwise the system program or the TR-8 itself may be destroyed.

Please install the extracted system program file into your TR-8 according to the following procedure.
  1. Hold down Pad [ 1 ] and switch on the power.
    INST select button BD through RC blink.
  2. Connect the computer to the TR-8's USB port by a USB cable.
    BD and Pad [ 1 ] blink.
  3. Open the "TR-8" drive folder in your personal computer.
  4. Copy TR8_UPD.BIN in the "tr8_sys_v161" folder into the "TR-8" drive by dragging and dropping it.
    * Do not copy the "tr8_sys_v161" folder.
  5. After copying the file is completed, remove the USB drive from the personal computer.
    With Windows
    Right-click on the "TR-8" icon in My Computer and execute “Eject.”
    With Mac
    Drag the "TR-8" icon to the Trash icon in the Dock.
  6. Disconnect the USB cable.
    SD and Pad [ 2 ] blink.
  7. Press the Pad [ 2 ] button to start updating the file.
    LT and Pad [ 3 ] blink and the SCATTER LEDs blink and light up in sequence, beginning with 1.
    The SCATTER LEDs light up from 1 to 10, and when BD through RC as well as pads 1 through 16 blink, the update has finished.
  8. Turn the TR-8 power off.
    Please verify the version number with the procedure described in "How to tell the version" part of this document.
