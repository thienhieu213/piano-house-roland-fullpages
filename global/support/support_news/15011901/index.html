Model name Compatibility Remarks
R-MIX Under
testing		 [ Important Notice ]
We have confirmed situations in which installation is not possible. For more information, refer to "Apology and Notice Regarding R-MIX Installation Failure." ( 2014.10.31 )

When the ".NET Framework 3.5 (includes .NET 2.0 and 3.0)" installation dialog box appears during the installation of R-MIX, click [Skip this installation] (This software framework is not required for the R-MIX operation.).

* When the OS is updated from Windows 8 to 8.1, R-MIX may not start. In such a case, uninstall R-MIX first and reinstall it.

* For the installation procedure, click here.
[Installation procedure]
To install R-MIX, refer to the following and the attached installation guide.
  1. Start your personal computer and logon as an administrator.
  2. After the desktop screen appears, click the [Explorer]button.
  3. Click [Computer] and set a software DVD-ROM on the DVD drive of the personal computer.
  4. Right-click the DVD-ROM icon shown in the Explorer screen, and click [Open].
  5. Double-click [Setup(.exe)] to start the installation. For the detailed procedure, refer to the attached installation guide.
