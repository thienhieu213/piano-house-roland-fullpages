Compatibility with macOS Ventura (macOS 13) (Intel Processor)

Compatibility with macOS Ventura (macOS 13) (Intel Processor)

Last updated: Jan 6, 2023
Our product's compatibility information with macOS Ventura (macOS 13) is as follows.
Compatibility with macOS Ventura (macOS 13) will be verified and announced after the release of macOS Ventura (macOS 13).
If you are using a Roland product with a connection to a Mac computer, we recommend that you stay with macOS Monterey (macOS 12) until the results of our verification process are published.

For compatibility information for BOSS brand products, please see here.
For compatibility information for Professional A/V category products, please see here.

* Please also check the System requirements for the audio/MIDI application and the operating system you're using.
* For the products introduced after macOS Ventura (macOS 13), please refer to each product information page on the Roland.com website.
This is to inform you about support for Mac computers using the Intel Processor.
* For information on support for Mac computers using the Apple Silicon, check here.
